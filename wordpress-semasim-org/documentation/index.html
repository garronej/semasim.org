Requirements

  • One USB 3G Dongle for each sim card you wish to use.

    • you can either use a Huawei 3G dongle from the following list, or you can purchase a ready to use Semasim dongle here.

  • A computer with a Linux installation

    • Supported Linux versions:
    - Debian/Raspian 7(jessie) or more recent
    - Ubuntu 14.04(trusty) or more recent

    Raspberry pi's are an ideal option. Follow our tutorial to set one up.
    It is equaly possible to run Semasim on Windows or MAC using a virtual machine. Instructions to set up a virtual machine using Virtual Box are given bellow.
    Orange pi single board computers are also suitable but require a few extra steps. Follow our guidelines to set one up.

Unlocking your Dongle:

- Download and install Putty

- Connect your 3G dongle to your Computer and follow the installation procedure.

- Open the Device Manager, expand the Modems tab, right click on HUAWEI Mobile Connect - 3G Modem and click on Properties. Select the Modem tab and copy the Port number.

- Run Putty and select serial under Connection type. Paste the port number under Serial line and press Open.

- Now you will see a blank screen. Type in atz and press enter then type in ati and press enter. you should see the following results.

- Copy the IMEI code from these results, go to the following Online calculator and submit the imei code to generate an unlock code.

- Finaly, in Putty, type in the following AT Command; AT^CARDLOCK="".

Enabling your Dongles voice features

You can download DC Unlocker. It does not require any installation. To use DC Unlocker you have to create an account and buy credits on dc_unlocker.com. enabeling voice functions on a 3G Dongle costs roughly 4 Euros.

- Run the DC Unlocker executable file that you just downloaded
- Enter login details from the DC Unlocker account you just created
- Click on the Check login button

- Connect your 3G GSM modem to your computer
- Click on the Detect modem button
- In the Unlocking tab, press the Activate Voice Button.

Installing Semasim server

Once you have your host machine set up and running, you simply need to open a Terminal and type in the Semasim installation command:
wget -nc -q -O - gw.semasim.com/install.sh | sudo bash
The program will then install and run itself automaticly.

Using Semasim

Now that you have a host machine with Semasim server up and runing and one or more suitable 3G Dongles you can follow the next guidlines and start using Semasim.

- Go to Semasim.com to create an account then login to your new account to access the Web application

- Insert the sim card that you want to use in to your 3G Dongle and connect the later to your host machine.

- To register your Sim card(s) your Host machine needs to have the same public ip address as the computer you are using for the web aplication. In other words they need to be connected to the same Wifi. Your sim card(s) will then be detected automaticly, just enter your pin code and give a name to your sim card to finalise the registration.

- Access the web phone. You will see all the sim cards that you have registered with ther contact lists. Just click on one of your contacts to make a call or send a text message. You can add or delete contacts as if you where on your phone.

Semasim administration tools

To Administer your Semasim server you need to access a Terminal on your host machine. You shall then be able to use the following commands.

  • Disable autorun of Semasim at the start of your host machine:
    sudo systemctl disable semasim
  • Enable autorun of Semasim at the start of your host machine:
    sudo systemctl enable semasim
  • Start Semasim:
    sudo systemctl start semasim
  • Stop Semasim:
    sudo systemctl stop semasim
  • Uninstall Semasim:
    sudo semasim_uninstaller run
  • List of connected 3G dongles:
    dongle list
  • Unlock sim card:
    dongle unlock -i your_dongles_imei -p sim_pin_code

To see all available command tools type in dongle --help.
If you need to access the Semasim log file, type in cat /usr/share/semasim/working_directory/log

If you want to administer your 3G dongles using AT commands, run asterisk_cli.sh:
sudo ./usr/share/semasim/working_directory/asterisk_cli.sh
You can send an AT command to any one of your 3G dongles by typing
dongle cmd device_name AT_command. Replace device_name with the name of the 3G dongle that you want to interact with. To see the list of connected 3G dongles and there names use dongle show devices.

Setting up your Raspberry pi

Download a Raspbian disk image
Fromatting your micro SD card
Write Raspbian image to SD card
Enable SSH server on your Rapsberry pi
Start your Raspberry pi and find its ip address
Connect to your Raspberry pi


Step 1: Download a Rapsbian disk image.
The Raspbian Buster lite version would be best, but the others will work as well.


Step 2: Formatting your micro SD card.
Start by downloading and installing SD card Formatter, then when you start it, select your SD card(it should be selected automaticly) and press the format button.


Step 3: Write Raspbian image to SD card.
Download and install balenaEtcher.
Start by selecting the Raspbian image that you downloaded earlier, then select your SD card and press the Flash button. Wait for the process to complete itself, it may take a few minutes.


Step 4: Enable SSH server on your Raspberry pi.
To be able to access and configure your Raspberry pi without having to use a keyboard and monitor, create a file named "SSH" with no extention at the root of your SD card.


Step 5: Start your Raspberry pi and find its ip address.
Connect your prepared SD card, power supply and network cable to your Raspberry pi.
Now you need your Raspberry pi's IP address.
You can find it by accessing your router's interface.
- Click on start.
- Type "cmd" in the search box.
- In the new window type in "ipconfig".
- Copy and paste the "Default Gateway" address in to your navigator.
- Login to your router.

Now you will be able to find your Raspberry pi's ip address in your router's DHCP allocation table or network map.


Step 6: Connect to your Raspberry pi.
If you are using linux, open a terminal and login to your Raspberry pi using SSH. Type in "ssh pi@the_ip_address_of_your_pi", then as the password, type in "raspberry". This password is set by default.

On Windows, download and install Putty to access your Raspberry pi. Under "Host Name", type in your Rapsberry pi's ip address, then to login use the default credentials:

login:pi
password:raspberry



You now have access to your raspberry pi and can install semasim server.

Setting up your virtual Machine on Windows

Download a Debian disk image
Create a virtual machine with VirtualBox
Boot your virtual machine from Debian disk image
Install Debian on your virtual machine
Debian configurations


Step 1: Download a Debian disk image.
It is recommended that you use the latest vertion available. you can allsow use Ubuntu but in this documentation we shall be working with Debian.


Step 2: Creating a virtual machine with VirtualBox
Download and install Virtual Box.

Once you have VirtualBox up and runing, click on the "new" button.

Now give a name to virtual machine and select Linux as the type of operating system. As the version, select Debian 64 bits or Debian 32 bits, depending on your processor. (If you are runnig on a 64 bits processor and "debian 64 bits" does not apear in the list check out the troubleshooting page to activate your processors virtualisation functions).

Select the amount of memory (RAM) you want to be allocated to your virtual machine. I recommend that you use a minimum of 1024 megabits, anything bellow that value would force your debian virtual machine tu run in low memory mode.

Now you must create your virtual hard drive. Select "Create a virtual hard disk now" then press the "Create" button.
Select "VDI" as the hard disk file type, then, for optimal performance, choose a fixed size for your storage.

Finaly, choose the size of virtual hard disk then press the "create" button. I recommend that you use at least 8 GB.

Your virtual machine is set up and ready to have Debian installed on it.


Step 3: Boot your virtual machine from Debian disk image.
First you must select your Debian virtual machine and go to "Settings.


When you are in "Settings" go to "Storage", select your IDE controller, choose the Debian disk image you downloaded previously as the "Virtual optical disk file" then press the OK button.


Step 4: Install Debian on your virtual machine.
Select your virtual machin and press the "Start" button to launch it.

Your virtual machine will then boot the Debian disk image. Select the "Install" option next.

You will then be guided through the installation stages such as language and regional settings and also the setting up of your credentials. When asked to enter a domain name, you can choose "Your_name.com" for example.
Do not use spaces in the name you want to use, if you want there to be a space, use underscore.

Next type in the name and password you wish to use. You can use the same password for the root and the user. Be sure to remember them, they will prove useful later on.
When your credentials have been succesfuly configured, you will be askes to choose a partitioning method; select "Guided: use an entire disk".


Then when asked wich disk you want to partition press "Enter". It will be your only option anyway.
As for the partitioning scheme, for simplicity's sake, put all files in one partition.


Finish partitioning and write changes to disk and Debian will start installing itself. When asked if you wish to scan another CD or DVD, answer NO.


Choose the country you are located in as the Debian archive mirror country and leave the preselected option as the Debian archive mirror.
Leave a blank in the HTTP proxy information.
When asked to choose which software to install, uncheck the "Debian desktop environement" option by high lighting it and pressing the space button. Having a desktop environment serves no purpose in this instance but if you wish to install it for other reasons, it should not cause any problems.

The installer will finaly suggest that you install the GRUB boot loader; Agree and select your virtual hard drive.
Your Debian installation is complete.


Step 5: Debian configurations.
Start Debian using the login and password you chose during the installation. While typing in the password it will seem that nothing is hapening, but this is normal, just type the password and press Enter.
You now have access to a Linux terminal.
First you need to enable administrator privileges; Type in the following command to go into root mode:
su
As the password, use the one that you set for root, during the Debian installation. Now install sudo by typing in the following command:
apt-get install sudo
Once sudo is installed you must Add your name to the list of administrators. Open the "sudoers" file by entering the following command:
nano /etc/sudoers
You must edit this file by writting your_user_name ALL=(ALL:ALL) ALL exactly underneeth the line where you can see
root ALL=(ALL:ALL) ALL. When you have finished, hold Ctrl and press x to exit. Press y to save and then press Enter to overite the existing sudoers file.


Restart your virtual machine.
